About Wyrdness

What is Wyrdness?

Wyrdness is a comprehensive, open-source database documenting 516+ paranormal phenomena from cultures around the world. Our mission is to preserve folklore, mythology, and unexplained mysteries with academic rigor and accessibility.

Our Approach

Each phenomenon in our database is meticulously documented with:

  • Verified Sources - Academic papers, historical texts, and credible eyewitness accounts
  • Cultural Context - Origin stories, regional variations, and cultural significance
  • Evidence Analysis - Critical evaluation of claims and documented sightings
  • Open Collaboration - Community-driven research via GitHub

Open Source & Open Data

Wyrdness is built on the principles of open knowledge. All data is freely available under Creative Commons licenses:

  • Data: CC0-1.0 (Public Domain)
  • Content: CC-BY-4.0 (Attribution Required)
  • Code: MIT License

Every phenomenon has a dedicated GitHub repository where you can view sources, report corrections, submit sightings, and contribute research.

API Access

Developers can access our data programmatically:

  • /api/v1/index.json - All phenomena
  • /api/v1/categories.json - Category breakdown
  • /api/v1/stats.json - Database statistics
  • /api/v1/{phenomenon-id}.json - Individual entries
